Unity Project 3: Ellen

This was an assignment done in second year Game Design. The prototype is set in an alien world. Ellen, the main character, has to navigate through a series of obstacles and enemies in order to reach the extraction point. The art assets in this prototype were downloaded from the Unity assets store. The implementation was done by me. This implementation includes the movement, animations, health, sound effects, user-interface and shooting mechanics for both the protagonist and the antagonists. A single level was created to display these mechanics effectively. The level includes locked-door puzzles, moving platforms, death traps, portals, as well as enemy AI. This assignment was created in order to give Game Design students an introduction on implementing platformer games.
