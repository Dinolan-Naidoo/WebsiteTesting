About me

I am an aspiring engineering professional looking to aid in the advancement of software development and data optimization. I am currently in my final year at the University of the Witwatersrand. I am studying a Bachelor of Engineering Sciences in Digital Arts. At the end of my degree I will have acquired a vast library of knowledge in software development, game design, as well core Electrical and Information Engineering concepts. A few of these concepts include Microprocessors, Electric Circuits, Electronics, Applied Physics, Electric and Magnetic Systems, as well as Engineering Mathematics.

I aspire to push the boundaries of what is technologically possible in digital design. I grew up in the digital era and this led me to have an enduring passion for it. Whether it’s the development of AAA games or data analytics for the corporate industry, I believe that I possess the skills and enthusiasm to pursue either. I am a diligent and goal-driven individual that excels both individually and in a team.

SOFTWARE EXPERIENCE

UNITY

2019 - Present

My degree is intended to produce graduates that can become specialist practitioners in the technological arena of software development within a game context. Game design is a core subject. The game design modules have a vigorous practical component, but also emphasise a solid academic grounding in the field. The main game engine used is Unity. Throughout the course I have created many games across a variety of genres such as First-Person Shooters, Simulations, Puzzles, Platformers and many more. I have acquired an extensive library of knowledge in character design, level design, puzzle design, user-interface, user-experience, and many more fundamental elements of games.

BLENDER

2019 - Present

Game Design IIB (course code: WSOA2009A) in my second year has equipped me with the fundamentals in using Blender for 3D graphic design. I have acquired the skills to create 3D models, use shader graphs, UV mapping, baking normal maps, rigging 3D models for complex behaviour, creating seamless animations and many more elements that Blender offers for graphical design. Furthermore, this course allowed me to gain the skills needed to import and use these 3D models efficiently within the Unity game engine.

TWINE

2019 - Present

Twine is an open-source tool created by Chris Klimas for making interactive fiction in the form of web pages. This software was used for one of my game design projects. Players read content and then interact by clicking links in the text. The software uses a flow chart structure that mainly involves scripting hyperlinks between the nodes. The software outputs an HTML file that allows the game to be played online. This project gave me an introduction to HTML and CSS. The insight gained by creating an interactive story game with HTML has given me a greater understanding of web design and creating games within the context of web pages.

MICROCHIP STUDIO

2020 - Present

The Microprocessors course (course code: ELEN2006A) in my second year gave me an introduction on using Microchip studio. Microchip Studio (formerly known as Atmel Studio) is a development environment that is used to write, build, and debug applications written in C/C++ or assembly code. In the Microprocessors course, I was tasked to build programs in assembly code that integrate with the ATMEGA328P microcontroller. Learning assembly code allowed me to gain a deep understanding of how data is transferred and manipulated via the use of the CPU(Central Processing Unit), RAM (Random Access Memory), ROM (Read-Only Memory), Internal Oscillator (the main timer of the MCU), the Data Bus and the input/output registers. I have learnt how to program basic industry standard microprocessor chips as well as make use of binary mathematics and sequential logic.

MY PORTFOLIO

Unity Project 1: Perfect Timing

This game is an experiment on the technical mechanics that make up game-play. The game aesthetics are purely black and white in order to eliminate any game-feel that arises from colourful aesthetics. Instead the focus is on how the main mechanic feels. The player must control a ball and navigate through the level.

Unity Project 2: Angry Birds Mod

This game was created to explore the technical side of coding an Angry Birds game. The game makes use of a slingshot mechanic that allows players to launch the protagonist into the antagonists in order to destroy them.

Unity Project 3: The Ranch

This game explores the core aspects of a farming simulation. The player takes control of a farm where they need to manage sheep and harvest crops in order to make a profit. A successful balance between buying and selling will produce revenue for developing a better farm.

Unity Project 4: Ellen Implementation

This prototype is a single level to demonstrate the implementation of the Ellen assets from the Unity store. The implementation includes movement mechanics, animations and level design.

Unity Project 5: Path-Finding Algorithm

This is a prototype to display a genetic machine learning algorithm implemented in C#. The agents learn to find an effiecient path over several generations in order to reach a target point.
